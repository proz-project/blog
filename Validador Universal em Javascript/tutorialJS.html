Tutorial - Validador Universal de Formulário

 

Objetivo: Este tutorial tem como objetivo explicar o funcionamento e o uso deste Validador de Formulários Web.

O Validador: Este validador é composto de funções que através de ER's (Expressões Regulares) formam um padrão para o preenchimento de um formulário.

Esta padronização trará benefícios tanto para o usuário quanto para o administrador do site:

  • o usuário não possuirá dificuldades para o preenchimento deste pois existem dicas e alertas que o ajudarão a completar este rapidamente;

    •
  • e o programador terá a segurança de que o formulário será totalmente preenchido, e ainda seguindo a forma correta e desejada por ele.

As funções: O formulário possui várias funções.Algumas são específicas de cada campo, outras possuem determinados objetivos como por exemplo barrar a digitação de números em campos específicos de letras (ex.: campo de nome), ou evitar que sejam digitadas letras em campos de CPF, Data, RG e CEP. Abaixo estão as funções usadas na composição do formulário:

  • function valida_campoVazio(campo);
  • function campo_radioCheckSelect(campo);
  • function limpaID(ID);
  • function informa(Ncampo);
  • function validar_nome(Ncampo);
  • function validar_CEP(Ncampo);
  • function validar_CPF(Ncampo);
  • function validar_RG(Ncampo);
  • function validar_email(Ncampo);
  • function validar_Data(Ncampo);
  • function valida_numeros(Ncampo);

Expressões Regulares: As Expressões Regulares são colocadas dentro de uma função. Ao final da digitação do usuário é feito uma comparação do que ele escreveu com o que a ER ordena, caso não esteja de acordo o programador poderá implementar uma dica ou instrução de como realizar o preenchimento daquele campo. CARACTERISTICAS DO VALIDADOR

- bloqueia digito de caracteres pré-definidos, nos campos numericos como CEP impede o digito de letras e em campos especifico de texto

como nome bloqueia o digito de números
- valida cep de acordo com o estado
- valida CPF
- valida Data
- valida senha com restrição minima de 3 caracteres
- auto completar nos campos: CEP,CPF,RG,Data
- valida campo radio, checkbox no sbmit do formulário

Modo de Usar o Validador: O usuário


EXEMPLO 1:
usando o validador em um campo nome.

<input type="text" id="nome" name="nome" onKeyPress="valida_digitos('nome')" onBlur="valida_campos('nome')">
no evento onKeyPress é chamada a função valida_digitos() passando como parametro o id do campo. A função valida_digitos() impede que seja digitado números no campo, quando o id passado for 'nome' ou qualquer outro configurado no arquivo validador.js

no evento onBlur é chamada a fnção valida_campos(), que quando passado como parametro 'nome' ele verifica se tem no mínimo 3 caracteres no campo

EXEMPLO 2:
usando o validador em um campo numérico, CEP,CPF,RG,DATA

<input type="text" name="rg" id="rg" onFocus="informa('rg')" onKeypress="valida_digitos('rg')" onKeyDown="formata('rg')" onBlur="valida_campos('rg')">
no evento onFocus é chamada a função informa() que exibe uma mensagem com descrição de como deve ser preenchido o campo na div com id='respostanomedocampo, a mensagem é de acordo com o parametro passado. Evento onKeypress é chamada a função valida_digitos(), a função impede que seja digitado letras no campo quando o parametro passado for cep,cpf,rg,data. Evento onKeyDown é chamada a função fomata(), a função da formatação no campo automaticamente de acordo com o parametro passado, no caso do exemplo cpf - xxxxxxxx-xx. Evento onBlur é chamada a função valida_campos() que verifica o número informado no campo, o numero é testado se ok então caixa de texto fica verde e o focus é liberado para outro campo, se não a caixa de texto fica vermelho e o focus permanece no campo.

OBS.: nos campos cep,cpf,data, o Evento onBlur a função chamada é valida_cep() - valida_cpf() - valida_data()
a validação do CEP é de acordo com o estado selecionado, por isso se for o usar a vlaidação de cep deve ter um campo onde seja tb informado o uf ou estado

EXEMPLO 3:

<form action="#" onSubmit="return valida_enviar(this)">
ao submeter o formulário é chamada a função valida_enviar(this), a função verifica se todos os campos do formulário foram preenhidos, se não é informada na div correspondente a cada campo uma mensagem, se sim o formulário é enviado.
